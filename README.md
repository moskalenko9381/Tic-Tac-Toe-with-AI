# Tic-Tac-Toe-with-AI
Tic-Tac-Toe with AI, using command line interface.
Commands:
  start
    Levels of AI: easy, medium hard
    User: user
  exit
  
You can play against another user or just look at game of two AIs (of the same level or different).
